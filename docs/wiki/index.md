# Bienvenue sur BAVI LEO 🏛️🤖

**BAVI LEO** — **B**ureau **A**gentique **V**irtuel **I**ntégré pour **L**EO — est votre système de bureaux IA spécialisés.

Chaque bureau est un assistant expert configurable, disponible via Telegram, qui combine les prompts métier de BAVI avec la puissance de Hermes Agent.

## Architecture

```
Telegram → BAVI LEO (Hermes Agent) → Skills spécialisés → LLM (DeepSeek/Ollama/Gemini)
                                       → Outils (web_search, fichiers, dashboards)
                                       → Cron workflows
```
