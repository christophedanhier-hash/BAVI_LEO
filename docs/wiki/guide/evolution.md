# 📖 Évolution du système

## Ajouter un bureau

1. Je crée le skill Hermes
2. J'ajoute la page wiki
3. Je mets à jour la navigation
4. Je push

## Améliorer un bureau

Un skill Hermes peut être modifié à tout moment. Le prompt système est versionné.

## Proposer un nouveau bureau

Tu as un besoin non couvert ? Dis-moi :
- Le nom du bureau
- Le domaine
- Les expertises nécessaires
- Le workflow souhaité

Je crée le skill en quelques minutes.
