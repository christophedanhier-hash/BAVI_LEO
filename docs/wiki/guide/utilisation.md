# 📖 Guide d'utilisation

## Lancer un bureau

Depuis Telegram, il suffit de dire :

```
bureau-robert : analyse le dossier de l'INAMI sur la digitalisation des flux
```

Ou plus simplement :

```
Bureau Robert, analyse le dossier INAMI
```

## Les formats de livrables

Chaque bureau produit un livrable structuré :

| Bureau | Format | Exemple |
|--------|--------|--------|
| Robert | Note structurée .md | Analyse IT + recommandations |
| Sophie | Business case .md | TCO/ROI, VAN, scenarii |
| AO | Relecture métier .md | Impacts AO |
| Gérard | Documentation .md | Manuel, procédure |
| Sylvie | Carnet de voyage .md | Itinéraire, journal |

## Combiner les bureaux

Demande une analyse multi-bureaux :

```
Bureau Robert avec lentille AO : analyse du projet X
```

## Archivage

Tous les livrables sont documentés dans le wiki BAVI LEO.
