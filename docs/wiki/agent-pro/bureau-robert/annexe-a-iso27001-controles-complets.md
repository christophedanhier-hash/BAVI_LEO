## 📝 STATEMENT OF APPLICABILITY (SoA)

La **Déclaration d'Applicabilité** est un document obligatoire qui liste chaque contrôle et indique :

| Colonne | Description |
|:--------|:------------|
| **ID** | Référence du contrôle (ex: 5.1, 6.2) |
| **Nom** | Nom du contrôle |
| **Applicable ?** | Oui/Non |
| **Justification** | Pourquoi applicable ou non |
| **État** | Implémenté / Partiellement / Non implémenté |
| **Référence** | Document de preuve associé |
